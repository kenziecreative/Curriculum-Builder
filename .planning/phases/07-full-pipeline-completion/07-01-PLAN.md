Generate the full metaskill map silently. Then run these five constraint enforcement steps (silent — user never sees corrections being made):

Step 1 — Duration scaling:
- `contact_hours` < 2: max 3 metaskill activations; at least 1 must be Exploring or Creating; all-six coverage rule waived
- `contact_hours` 2–16: 4–5 metaskills
- `contact_hours` > 16: all six required

Step 2 — Developability hierarchy:
Assign `sequence_position` values. Verify: first Innovating sequence_position > first Exploring AND first Creating. Verify: first Adapting sequence_position > first Exploring AND first Creating. If violated: reorder sequence_positions to satisfy constraint without changing content.

Step 3 — Thinking routine specificity:
For each `activation_activity`: is it a specific named routine? Prohibited patterns: "encourage", "discussion", "reflection", "brainstorm", "debrief", "explore", "discuss". Acceptable examples: "See-Think-Wonder", "Claim-Support-Question", "Pre-mortem", "I Used to Think / Now I Think", "Connect-Extend-Challenge", "Five Whys", "Charette Protocol", "SCAMPER". If generic: replace with a named routine appropriate to the metaskill and content domain. Record change.

Step 4 — Transfer prompt grounding:
For each `transfer_prompt`: must reference the specific `transfer_context` from Stage 1 (name a real work situation, artifact, or decision the learner will face). Generic encouragement phrases like "apply this to your work" are prohibited. If generic: regenerate with specific reference. Record change.

Step 5 — Imagining conditional fields:
For any record with metaskill_name = Imagining: set `evidence_gap_acknowledgment: true`. Set `imagining_adjacent_practice` to one of: "scenario planning", "futures thinking", or "mental simulation" (pick most appropriate for content domain).

**Gate display (after constraint enforcement):**
Show the full metaskill activation map. Format per metaskill record:

```
**[Thinking Skill Name]**
Activity: [activation_activity — named routine]
When: [session_reference plain language, e.g., "Session 2 — Application block"]
Real-work connection: [transfer_prompt — specific real-work situation]
[If Imagining: Note: "This thinking skill has limited research evidence — the adjacent practice ([imagining_adjacent_practice]) is used to build the skill indirectly."]
```

Then the gate:

Does this activation plan match what you want for this program?
