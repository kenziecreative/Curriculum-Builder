**1. Add a "Research Document Recognition" section after the "Context Received" section.**

Content: When source materials include research documents (skill decompositions, misconception inventories, expert-novice difference analyses, practitioner workflow maps, transfer barrier studies, cognitive task analyses), the auditor should:

- Recognize these as research content from the material itself — the user does not label them
- Recognition signals: academic citations, systematic categorization of skills/errors/misconceptions, explicit expert-novice comparisons, structured workflow documentation, empirical findings about learning transfer
- Research content counts toward extraction_confidence the same way any source material does — a skill decomposition that explicitly states outcomes is High confidence, same as a facilitator guide that does
- Same confidence bar applies: High still means "explicitly stated in source material"

**2. Add a "Research Insight Extraction" section after the quality rubric.**

Content: After completing the standard per-stage assessment, scan all source materials for non-schema research insights — content that does not map to a specific stage's required fields but would help downstream generators produce better curriculum.

Research insight categories (extract if found):
- **Misconception inventories** — common errors, misunderstandings, or naive mental models learners bring. Relevant to: Session Content (pre-work addressing misconceptions), Assessment Design (distractors based on real misconceptions)
- **Expert-novice differences** — what experts do that novices don't, skill progression markers. Relevant to: Learning Outcomes (progression calibration), Module Structure (sequencing decisions)
- **Transfer barriers** — documented obstacles to applying learning in real work. Relevant to: Transfer Ecosystem (designing around known barriers), Session Content (practice scenarios that address barriers)
- **Practitioner workflow maps** — how the skill is actually used in work context. Relevant to: Session Content (realistic practice activities), Assessment Design (authentic assessment scenarios)
- **Cognitive load indicators** — complexity hotspots, prerequisite chains, common confusion points. Relevant to: Module Structure (pacing), Session Content (scaffolding decisions)

For each insight found, record: the insight (one sentence), which category it falls into, which downstream stage(s) it is relevant to, and the source document it came from.
