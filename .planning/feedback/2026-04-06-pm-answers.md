# PM Clarification Answers

**Source:** Kasie Treadway
**Date:** 2026-04-06
**Context:** Follow-up answers to builder team questions about Boost Camp feedback

---

## 1. Teaching content layer scope
Full prose voiceover scripts preferred, not talking point outlines. Should be grounded in weekly written guides and supplemental materials the agent already generates.

## 2. Case examples
Hypothetical only. Must reflect Hello Alice audience: women entrepreneurs, POC, immigrants, underrepresented founders. Industries: food, beauty, retail, services. Not generic tech startup archetypes.

## 3. Brand assets as input
Hello Alice brand guidelines baked in as default — not uploaded or asked about during intake. Only intake question: partner-specific branding for co-branded programs.

## 4. Format for new deliverables
- Internal Weekly Content Map: Word document
- Curriculum Completion Tracker: Spreadsheet (Excel/Google Sheets)

## 5. "Needs review/update" determination
Agent pre-populates status column. Hello Alice team does final review and sign-off.

## 6. Business Health Score
Hello Alice's standard pre-program assessment, maintained outside the builder. Baked-in default for Boost Camp builds — agent references it, doesn't generate it.

## 7. Tone calibration
Tool is Hello Alice-only. Brand voice is a fixed constant, not a variable. No voice-capture step or multi-org brand system needed. Revisit if tool extends beyond Hello Alice.
