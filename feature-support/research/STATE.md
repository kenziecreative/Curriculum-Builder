DCR validated (d=0.50 analogical encoding) but needs novice scaffolding + Validate step. **CONTRADICTED: doctrine designs for individual learners; cooperative learning d=0.54-0.78 — social learning layer required, not optional.** Transfer ecosystem (pre/in/post-program) is the architectural response to 7 documented failure modes. Open vs. closed skill classification needed. Community seeding (not completion) for compressed formats.
- Phase 9: **Gap confirmed decisively.** No existing tool across 3 market categories encodes pedagogical constraints as structural generation requirements. Quality gap is documented and systematic (78% lesson plans need adjustment, 40% fabricated references, 32% assessment diversity reduction). Critical finding: generic AI adoption = no measurable impact on ID work; research-informed AI workflows = 60-80% efficiency gains. The differentiator is the constraint architecture, not the AI model. "Jagged frontier" maps what to automate, constrain, keep human. Claude Code plugin niche genuinely unoccupied. Three emerging frameworks (ADGIE, ARCHED, FRAME) all confirm human expertise + AI constraint = viable pattern.
- Phase 10: **Architecture defined.** Claude Code plugin with bundled skills (one per pipeline stage), subagents (validation + parallel generation), hooks (enforcement gates). Nine-stage generation pipeline: intake → outcomes → assessments → modules → sessions → metaskills → transfer → marketing → validation. Three-layer doctrine encoding: CLAUDE.md (constraint summary), schemas (binding constraints), reference files (design knowledge). Three-tier validation: schema (automated), rubric (automated + human review), transfer ecosystem (human-driven). MVP is five stages: intake, outcomes, assessments, modules, validation. Output format makes pedagogical grounding visible and auditable. Ten architectural principles derived from Phases 1-9.
- Phase 11: **Validation criteria defined.** Three-tier validation confirmed by empirical evidence. Tool must construct its own rubric from QM alignment + Merrill FPI (TALQ) + backward design + Hu et al. AI dimensions — no existing instrument covers the full range. Transfer design is the least scored element across all major rubrics (gap = opportunity). Pairwise blind comparison is the methodological standard for head-to-head evaluation. Tessmer's expert review (4 sub-analyses) is the canonical short-cycle method. Bloom's classification automatable at 94%; scaffolding quality requires humans. LLMs show poor intra-rater consistency — validation agent should produce reports, not certifications. Eight testable success criteria defined (6 automatable, 2 human). TALQ scales map directly to schema-enforceable session/module fields.
