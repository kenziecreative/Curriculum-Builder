name: Project North Star
description: The goal of this research is to build a well-informed curriculum engine for adult learning, grounded in learning science — not to validate a market opportunity.
type: project
---

The end goal is an engine (Claude Code tool) that builds adult learning curriculum. The research exists to make sure that engine is well-informed — that when it generates curriculum for any topic and any audience, the output reflects what learning science actually says works.

**Why:** The existing accelerator prompt template produced marketing-heavy, pedagogically thin output. The researcher wants the replacement to be substantively grounded, not just polished.

**How to apply:** Every phase of research should be evaluated against this question: does this finding make the engine smarter about how adults actually learn? Market viability, positioning, and audience-specific verticals are secondary. The primary output is a well-informed tool, not a validated product.
